Halo, nama saya **Kelfin Maulana**. Saya adalah seorang **Front End Developer dan Project Manager** dengan pengalaman lebih dari 2 tahun sebagai Project Manager. Saya memiliki latar belakang pendidikan **Bachelor of Computer Science (S.Kom)** dari Institut Teknologi Indonesia. Saya memiliki pengalaman kerja yang beragam, mulai dari magang di PT. Visionet Data Internasional sebagai Call Center Provider hingga posisi saat ini sebagai Project Manager di PT. Bougenville Griya Persada. Selain itu, saya juga aktif dalam organisasi, di mana saya pernah menjabat sebagai Ketua Departemen Pengembangan Sumber Daya Manusia di Himpunan Mahasiswa Teknik Informatika Institut Teknologi Indonesia. Keahlian saya meliputi penggunaan **Microsoft Word, Microsoft Office, Query MYSQL, JavaScript**, dan **PHP Framework Laravel**. Saya juga memiliki kemampuan soft skill seperti kepemimpinan, kerja sama tim, komunikasi efektif, dan manajemen waktu. Di luar pekerjaan, saya memiliki minat dalam mendaki gunung dan bercita-cita untuk mendaki 7 summit di Indonesia. Saya juga berambisi untuk memiliki perusahaan sendiri di bidang konstruksi dan teknologi. Saya berharap dapat terus mengembangkan kemampuan saya dan berkontribusi lebih banyak lagi di dunia teknologi dan manajemen proyek.
